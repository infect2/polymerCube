# polymerCube
Polymer based CSS3D based Cube View

How to use
It is as easy as below,
    <cube-element mode="flat" imgSrc='{
        "left": "http://url2image",
        "top": "http://url2image",
        "bottom": "http://url2image",
        "front": "http://url2image",
        "right": "https://url2image",
        "back": "http://url2image"
    }'>
    </cube-element>
